Условия использования

1. Общие положения

Используя сервис FACEIT Stats, вы соглашаетесь с настоящими условиями использования. Сервис предоставляет доступ к статистике игроков FACEIT через их Steam URL.

2. Условия использования сервиса

Для использования сервиса необходимо:

  • Иметь действующий аккаунт FACEIT
  • Предоставить корректный Steam URL
  • Соблюдать правила использования API FACEIT

3. Ограничения использования

Запрещается:

  • Использовать сервис для массового сбора данных
  • Нарушать правила FACEIT и Steam
  • Использовать сервис для вредоносных целей
  • Пытаться обойти ограничения API

4. Интеллектуальная собственность

  • Логотип FACEIT принадлежат FACEIT Ltd
  • Логотип Steam принадлежат Valve Corporation
  • Дизайн и код сервиса принадлежат нам

5. Ограничение ответственности

Мы не несем ответственности за:

  • Точность данных, полученных через API FACEIT
  • Работу сторонних сервисов (FACEIT, Steam)
  • Потерю данных при сбоях в работе API

6. Изменения в условиях

Мы оставляем за собой право изменять условия использования. Продолжение использования сервиса после изменений означает принятие новых условий.

Мы можем прекратить доступ к сервису в случае:
