Политика конфиденциальности

1. Сбор информации

Наш сервис FACEIT Stats собирает следующие данные:

  • Steam URL пользователя для поиска профиля FACEIT
  • Публичные данные профиля FACEIT (никнейм, аватар, статистика)
  • История матчей и статистика игр

2. Использование информации

Собранная информация используется исключительно для:

  • Отображения статистики игрока
  • Предоставления истории матчей
  • Улучшения работы сервиса

3. Хранение данных

Мы не храним персональные данные пользователей. Вся информация отображается в реальном времени через API FACEIT.

4. Безопасность

Мы принимаем меры для защиты информации:

  • Используем защищенное HTTPS-соединение
  • Не храним API ключи на клиентской стороне
  • Регулярно обновляем систему безопасности

5. Cookies

Мы используем только необходимые cookies для:

  • Сохранения выбранной темы оформления
  • Сохранения выбранного языка интерфейса

6. Сторонние сервисы

Наш сервис интегрирован со следующими API:

  • FACEIT API - для получения статистики игроков
  • Steam API - для поиска профилей по Steam URL

7. Права пользователей

Пользователи имеют право:

  • Не предоставлять свой Steam URL
  • Очистить cookies браузера
  • Связаться с нами по вопросам конфиденциальности

8. Изменения в политике

Мы оставляем за собой право вносить изменения в политику конфиденциальности. Все изменения будут опубликованы на этой странице.
